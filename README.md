예)치지직 → !치지직<br>
7. 띄어쓰기는 없에고 입력<br>
예)!가덕도신공항 건설을 위한 특별법 → !가덕도신공항건설을위한특별법
8. 숫자, 영어, 일본어, 중국어, 특수 문자 등 한국어가 아닌 글자가 포함되어 있는 단어는 제외<br>
예)!112신고의 운영 및 처리에 관한 법률 → 숫자가 있기에 입력 불가<br>
   !디어♥보컬리스트 → 특수문자(♥)가 있기에 입력 불가.<br>
10. 지하철 역명은 역을 표현하는 "역" 글자와 부역명을 제외하고 입력 가능<br>
예)!가산디지털단지역(마리오아울렛) → !가산디지털단지 로 입력<br>
